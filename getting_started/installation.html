RAIN Installation¶

In this document you will find all the information required for configuring RAIN development machine.

Nodejs server¶

Linux / Unix / Mac OS¶

Before you start to install node make sure you have the following tools installed:
  1. unzip (Ubuntu: sudo apt-get install unzip)
  2. gcc (Ubuntu: sudo apt-get install gcc)
  3. g++ (Ubuntu: sudo apt-get install g++)
  4. openssl (Ubuntu: sudo apt-get install openssl)
  5. libssl-dev (Ubuntu: sudo apt-get install libssl-dev)
  6. make (Ubuntu: sudo apt-get install make)

Now you can start installing node.

  1. wget http://nodejs.org/dist/v0.6.13/node-v0.6.13.tar.gz
  2. tar -zxvf node-v0.6.13.tar.gz
  3. cd node-v0.6.13
  4. chmod u+x configure
  5. chmod u+x tools/waf-light
  6. ./configure && make && sudo make install
  7. Make sure /usr/local/bin is added to your PATH environment variable.
  8. node -V to make sure node can be run
  9. sudo npm install node-gyp -g to install the latest node native module build tool

Installing via package manager:

https://github.com/joyent/node/wiki/Installing-Node.js-via-package-manager

Windows OS¶

  1. Download http://nodejs.org/dist/v0.6.13/node-v0.6.13.msi
  2. Install the msi package
  3. That’s all :)
  4. NPM is in the bundle too ;)

Mac OS¶

On Mac OS operating systems you need to execute the following commands:

  1. Download http://nodejs.org/dist/v0.6.13/node-v0.6.13.pkg
  2. Install the package
  3. That’s all :)
  4. NPM is in the bundle too ;)
  5. npm install node-gyp -g as administrator to install the latest node native module build tool

RAIN SDK¶

Linux / Unix / Mac OS¶

The easiest way to configure your development machine is to use RAIN SDK. The SDK was currently tested on Ubuntu Linux and on Mac OS X. Below you can find the commands to get the sdk running on Unix / Linux operating systems ( npm is required! ):

  1. sudo npm install node-gyp -g
  2. sudo npm install rain -g

If everything went well you should be able to run rain sdk from command line. For more information about RAIN sdk creating first project read: RAIN SDK Tutorial

RAIN from source code¶

This is the way to go if you want to contribute to this project. To get started with this you need to execute the following commands:

Linux / Unix / Mac OS / Windows¶

  1. git clone https://github.com/rainjs/rainjs.git
  2. cd rainjs
  3. sudo npm install node-gyp -g
  4. npm install -d
  5. sudo npm link
  6. rain start

Test installation¶

This should give you a running instance of RAIN server and mothership. For testing your installation try to access: http://localhost:1337/example/index.
