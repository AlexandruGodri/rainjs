Class DataLayer¶

This module is used to call the server-side functions.

Constructor¶

class DataLayer()¶

Methods¶

loadData¶

DataLayer#loadData(componentOpt, callback)¶
Arguments:
  • componentOpt (Object) – the component information
  • componentOpt.id (Object) – the component id
  • componentOpt.version (Object) – the component version
  • componentOpt.viewId (Object) – the view id
  • callback (Function) – receives the error and data as parameter

Validates all necessary parameters and invokes the data function from the component to receive custom data. If the data returns, it calls the callback function.
