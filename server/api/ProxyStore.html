Class ProxyStore¶

Proxy store implementation for connect. Uses dependency injection to aggregate implementations for local and remote stores. By default, a MemoryStore() is used for the local store and a MemoryStore() is used for the remote store. It uses lazy instantiation of stores.

It exposes the ProxyStore#authorize() method for switching the store implementation at runtime, i.e. providing a way to switch the store at runtime after login.
