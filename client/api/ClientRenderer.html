Class ClientRenderer¶

Handler for asynchroniously rendering components in rain. This is achived by sending each component individually to the client as soon as it’s available as a JSONP call to the renderer. This way the user gets a more responsive experience, since every comonent gets rendered as soon as it’s available, thus giving quicker visual feedback to the user.
