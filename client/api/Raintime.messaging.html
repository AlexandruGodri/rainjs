subscribe¶

Raintime.messaging.subscribe(eventName, callback, viewContext)¶
Arguments:
  • eventName – Event name we want to subscribe to. Can be any string value.
  • callback –
    This is the callback method that will get executed. It must have
    a single parameter called data.

    Ex: function(data)

  • viewContext – the ViewContext of the component publishing the event

This is the method that allows registration of a callback method to a desired event.
