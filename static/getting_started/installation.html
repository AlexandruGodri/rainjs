RAIN Installation¶

In this document you will find all the information required for configuring RAIN development machine.

Nodejs server¶

Linux / Unix / Mac OS¶

Before you start to install node make sure you have the following tools installed:
  1. unzip (Ubuntu: sudo apt-get install unzip)
  2. gcc (Ubuntu: sudo apt-get install gcc)
  3. g++ (Ubuntu: sudo apt-get install g++)
  4. openssl (Ubuntu: sudo apt-get install openssl)
  5. libssl-dev (Ubuntu: sudo apt-get install libssl-dev)
  6. make (Ubuntu: sudo apt-get install make)

Now you can start installing node.

  1. wget https://github.com/joyent/node/zipball/v0.4.12 -O node-v0.4.12.zip
  2. unzip node-v0.4.12.zip
  3. cd joyent-node-41e5762
  4. chmod u+x configure
  5. chmod u+x tools/waf-light
  6. ./configure && make && sudo make install
  7. make sure /usr/local/bin is added to your PATH environment variable.
  8. node - to make sure node can be run
  9. curl http://npmjs.org/install.sh | sh

Windows OS¶

  1. Install cygwin with the following options checked for install + Base + Database + Devel + Editors + Mingw + Net (make sure curl is checked in) + Shells + Python + Perl
  2. Run cygwin
  3. Edit /etc/resolv.conf file and add your nameservers like: nameserver <ip>
  4. Download node from https://github.com/joyent/node/zipball/v0.4.12 -O node-v0.4.12.zip
  5. Extract node in your home folder from cygwin.
  6. Enter node folder
  7. ./configure && make && make install
  8. node - to make sure node can be run
  9. In the root folder of your node workspace execute:
  10. git clone https://github.com/cspotcode/npm.git
  11. cd npm
  12. ./configure && make && make install — This will take a while so you can go grab a cup of coffee.
  13. npm -v

Redis Server¶

For both RAIN SDK or RAIN from source code you need redis server.

Ubuntu OS¶

On Ubuntu is really simple:

sudo apt-get install redis-server

Mac OS¶

On Mac OS operating systems you need to execute the following commands:

  1. wget http://redis.googlecode.com/files/redis-0.900_2.tar.gz
  2. tar xzvf redis-0.900_2.tar.gz
  3. cd redis-0.900
  4. make

I recommend you do not install this globally. It is better if you simply run the following command from redis-0.900 folder:

  1. ./src/redis-server

Windows OS¶

  1. Download redis file https://github.com/downloads/dmajkic/redis/redis-2.4.2-win32-win64-fix.zip
  2. Unzip the file.
  3. cd redis-2.4.2-win32-win64-fix/32bit/
  4. ./redis-server.exe

Please execute this command everytime before actually running mothership and rain.

RAIN SDK¶

Linux / Unix / Mac OS¶

The easiest way to configure your development machine is to use RAIN SDK. The SDK was currently tested on Ubuntu Linux and on Mac OS X. Below you can find the commands to get the sdk running on Unix / Linux operating systems:

  1. curl http://npmjs.org/install.sh | sudo sh
  2. Execute npm -version to make sure npm is correctly installed.
  3. sudo npm install rain -g

If everything went well you should be able to run rain sdk from command line. For more information about RAIN sdk creating first project read: RAIN SDK Tutorial

Windows Comments¶

For RAIN SDK to run on Windows please run the following commands:

  1. cd /home/<your user folder>
  2. mkdir node_kit
  3. cd node_kit
  4. Copy content from Install.sh content under install.sh and run it into node_kit folder
  5. npm install rain@0.4.0
  6. export PATH=$PATH:/home/<your user folder>/node_kit/node_modules/rain

Now everything is ready for you. You can execute rain on command line. To improve the experience for later use edit your .profile file and add as last line the following command:

export PATH=$PATH:/home/<your user folder>/node_kit/node_modules/rain

Like this you don’t have to execute export command each time you launch cygwin.

RAIN from source code¶

This is the way to go if you want to contribute to this project. To get started with this you need to execute the following commands:

Linux / Unix / Mac OS / Windows¶

  1. git clone https://github.com/rainjs/rainjs.git
  2. cd rain
  3. npm install -d
  4. Make sure redis server is running.
  5. node node_modules/rain-mothership/run.js
  6. node run.js

Test installation¶

This should give you a running instance of RAIN server and mothership. For testing your installation try to access: http://localhost:1337/components/cockpit/htdocs/main.html
